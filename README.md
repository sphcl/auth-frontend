# auth-frontend
Este repositório contém a interface de um Captive Portal desenvolvido para fins acadêmicos e demonstrativos. O projeto inclui um formulário de autenticação com validação, modo claro/escuro, logos institucionais e página de sucesso pós-login.

#### Tecnologias Utilizadas
#####•HTML5
#####•CSS3
#####•JavaScript

####Funcionalidades
#####•Formulário de login
#####•Redirecionamento para página de sucesso após submissão
#####•Alternância entre modo claro e escuro
#####•Layout responsivo
#####•Logos institucionais padronizadas
#####•Organização visual aprimorada usando Flexbox
